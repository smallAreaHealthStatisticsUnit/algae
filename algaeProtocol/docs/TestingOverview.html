Testing Strategy

by Kevin Garwood

The ALGAE protocol has been successfully tested with a suite of 300 test cases that compare results created by the software with results created by hand. ALGAE compares the two and generates a report showing which ones passed or failed. An example of the test run is here.

Our test plan was implemented by exercising a sequence of six test areas. Click on a test area to learn more about it:
