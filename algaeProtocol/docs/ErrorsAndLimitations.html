Errors and Limitations

by Kevin Garwood

Renaming early life stage result file

The file results_early_life_stages_[Date stamp].csv was renamed to res_early_life_stages_[Date stamp].csv to make the file comply with result file naming conventions used for all other result files.

Shifted the numbering of variables that compare exposure measurements

The names of variables in the algae41 series have been changed so that it starts with algae4100_person_id instead of algae4100_life_stage. The change was made to be consistent with having person_id considered a variable in other variable groups.

The same change was made to variables in the algae11, algae12, algae21, algae22, algae37, algae38, algae42, algae43, algae44, algae45, algae46 series as well.

Variables of the form algae[code area]00_person_id were inserted in the algae51, algae52, algae61, algae62 series.

Corrected spellings of variable names

Variables in the algae42 and algae46 series were changed from aglae... to algae.... aglae48 to algae48 aglae42

Corrected name of result file for comparing life stage mobility vs birth address assessments

A search and replace error was corrected to change the result table and result CSV from the form results_early_stg_mob_vs_stg_mob_birth to the form res_early_mob_uncln_vs_no_mob_birth
