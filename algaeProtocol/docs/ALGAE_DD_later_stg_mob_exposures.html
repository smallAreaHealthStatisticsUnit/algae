ALGAE Data Dictionary: Later life exposures, using life stage mobility assessment (algae3700-algae3716)

by Kevin Garwood

Context of Variables

These later life exposure values are based on the life stage mobility assessment method. The assessment uses the location study members occupied on the first day of each life stage to represent their location for all of that life stage. It ignores exposure contributions for all other locations study members may have used during their exposure period.

Cumulative, average and median exposures are calculated for each pollutant (NAME, NOX_rd, PM10_rd, PM10_gr, PM10_tot) for each life stage (T1, T2, T3, EL) for each person.

Location of Result File

You will find these variables in a file having a name that fits the form: 
res_later_stg_mob_exp_[Date stamp].csv
which will be found in the directory: 
later_life/results/exposure_data/mobility_life_stage
or 
later_life/results/exposure_data/mobility_life_stage

Variable Naming Conventions

It may be quicker to understand the variables through naming conventions rather than looking at specific table entries. The basic format of variables in this section follows this pattern: 
	algae37[0-16]_[pollution type]_[aggregate value]

In this pattern:

  • algae37: indicates that they refer to exposure values for later life that are based only on the locations that study members occupied on the first day of each life stage.
  • pollution_type: will be name, nox_rd, pm10_tot, pm10_rd, pm10_gr
  • aggregate value: sum for cumulative value, avg for average value and med for median value.

The pollutant codes have the following meanings:

  • name: high level pollution that comes from outside the exposure area
  • nox_rd: Nitrogen oxide pollution coming from roads
  • pm10_rd: PM10 particulate matter coming from roads.
  • pm10_gr: PM10 particulate matter coming from sources other than roads.
  • pm10_tot: PM10 particulate matter coming from either roads or other sources.

Variable Dictionary

Variable Description
algae3700_person_id An anonymised or pseudonymised identifier which represents a study member. ALGAE uses this variable to link data together for a given study member.
algae3701_life_stage The name of a life stage. For example, "T1" may be the name of the Trimester 1 life stage.
algae3702_name_sum Cumulative exposure of NAME for the given life_stage.
algae3703_name_avg Average exposure for NAME measured for the given life_stage
algae3704_name_med Median exposure for NAME measured for the given life_stage
algae3705_nox_rd_sum Cumulative exposure of NOX (road sources) for a given life_stage.
algae3706_nox_rd_avg Average exposure of NOX (road sources) for a given life_stage.
algae3707_nox_rd_med Median exposure for NOX (road sources), measured for the given life_stage
algae3708_pm10_gr_sum Cumulative exposure for PM10 (non-road sources), measured for the given life_stage
algae3709_pm10_gr_avg Average exposure for PM10 (non-road sources), measured for the given life_stage
algae3710_pm10_gr_med Median exposure for PM10 (non-road sources), measured for the given life_stage
algae3711_pm10_rd_sum Cumulative exposure for PM10 (road sources), measured for the given life_stage
algae3712_pm10_rd_avg Average exposure for PM10 (road sources), measured for the given life_stage
algae3713_pm10_rd_med Median exposure for PM10 (road sources), measured for the given life_stage
algae3714_pm10_tot_sum Cumulative exposure for PM10 (all sources), measured for the given life_stage
algae3715_pm10_tot_avg Average exposure for PM10 (all sources), measured for the given life_stage
algae3716_pm10_tot_med Median exposure for PM10 (all sources), measured for the given life_stage
